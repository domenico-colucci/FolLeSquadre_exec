**************ISTRUZIONI**************
- Scaricare il contenuto della cartella Web in un cartella locale
- Da linea di comando, dopo essersi spostati in tale cartella dare il seguente comando:

python3 -m http.server

- In risposta il terminale darà la disponibilità di un link del tipo http://0.0.0.0:8000/ il quale, copiato e incollato su un browser dà accesso alla lista dei file
- Da tale lista cliccare sull'unico file html presente (folLeSquadre.html)
